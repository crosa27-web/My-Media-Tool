Developer Setup Required

To make this app functional, you must insert your Google Cloud credentials into the JavaScript code.

  1. Open this HTML file in a text editor.
  2. Scroll down to the <script> section.
  3. Locate the CLIENT_ID and API_KEY variables.
  4. Paste your credentials from the Google Cloud Console.
